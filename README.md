# iac

## Install this collection

```bash
ansible-galaxy collection install jonsible.iac
```

## Bootstrap a machine

```bash
ansible-playbook jonsible.iac.bootstrap -i inventory.yaml
```

## Harden a machine

```bash
ansible-playbook jonsible.iac.bootstrap -i inventory.yaml
```

## Setup KVM server

```bash
ansible-playbook jonsible.iac.kvm -i inventory.yaml
```
